# Чеклісти для тестування

## Модуль: Створення та додавання пари в розклад

### Функціональність

- Відкривається форма створення пари
- Можна обрати викладача зі списку
- Можна обрати предмет
- Можна обрати тип заняття
- Можна вказати “Год”
- Працює чекбокс “Групова”
- Можна вказати онлайн-заняття
- Можна ввести “Предмет для сайту”
- Кнопка “ЗБЕРЕГТИ” створює нову пару
- Створена пара відображається у списку доступних пар
- Відкривається модуль розкладу
- Можна обрати групу для редагування розкладу
- Відображається таблиця розкладу вибраної групи
- Можна перетягнути створену пару у вільний часовий слот
- Після перетягування відкривається вікно вибору аудиторії
- Можна обрати аудиторію зі списку
- Кнопка “Вибрати” додає пару у розклад
- Після підтвердження пара відображається у розкладі
- Кнопка “ОЧИСТИТИ” очищає всі поля форми
- Дані зберігаються після оновлення сторінки
- Система перевіряє зайнятість аудиторії
- Система перевіряє конфлікти розкладу

---

## Негативні сценарії

- Неможливо зберегти форму без вибору предмета
- Неможливо зберегти форму без вибору викладача
- Неможливо зберегти форму без вибору типу заняття
- Відображається помилка при незаповненому полі “Год”
- Система не дозволяє створити пусту пару
- Неможливо перетягнути пару у зайнятий слот
- Відображається повідомлення про конфлікт слоту
- Відображається помилка при виборі зайнятої аудиторії
- Неможливо підтвердити створення пари без вибору аудиторії
- Неможливо створити дубльовану пару в одному слоті
- Некоректні дані не зберігаються
- Помилки валідації зникають після правильного заповнення
- Після натискання “Закрити” створення пари скасовується

---

## UI/UX

- Поля форми мають зрозумілі назви
- Випадаючі списки відкриваються коректно
- Кнопки “ЗБЕРЕГТИ” та “ОЧИСТИТИ” помітні та доступні
- Кнопка “Вибрати” у вікні аудиторій працює коректно
- Повідомлення про помилки відображаються біля відповідного поля
- Конфліктні слоти візуально позначаються
- Вільні слоти доступні для drag-and-drop
- Перетягування пари працює плавно та без зависань
- Інтерфейс не ламається при зміні розміру вікна
- Форма коректно відображається на різних екранах
- Після очищення форми всі поля повертаються до початкового стану
- Після додавання пари розклад автоматично оновлюється
